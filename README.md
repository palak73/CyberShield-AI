# 🛡 CyberShield AI

AI-Powered Cyber Crime Complaint Analyzer for Police Cyber Cells

---

## 📌 Overview

CyberShield AI is a lightweight cybercrime complaint triage and analysis system designed to assist law enforcement agencies and cybercrime cells in handling cyber fraud complaints efficiently.

The system automatically:
- Detects scam categories
- Assesses threat severity
- Extracts digital evidence
- Maps applicable cyber laws
- Generates police-style incident summaries
- Provides cyber safety recommendations

---

## 🚨 Problem Statement

Cybercrime complaint portals and police cyber cells receive thousands of complaints daily related to:

- OTP fraud
- Phishing attacks
- UPI fraud
- Fake loan scams
- Sextortion
- Fake job scams
- Investment scams

Manual complaint analysis is time-consuming and delays incident response.

CyberShield AI aims to automate preliminary complaint triage and improve cyber incident response efficiency.

The project uses:

- Rule-Based NLP
- Keyword Intelligence
- Regex-Based Evidence Extraction
- Threat Severity Scoring
- Cyber Law Mapping

instead of heavy machine learning models to ensure:
- Fast execution
- Explainable results
- Lightweight deployment
- Practical usability for cyber cells
